# 🔐 AuthentiCampus AI  
### Consent-First Synthetic Identity & Deepfake Risk Detection for Campuses

AuthentiCampus AI is a privacy-first AI-powered verification system designed to protect students and institutions from emerging synthetic identity fraud, deepfakes, and manipulated digital content.

Built for the **AI + Cybersecurity & Privacy** theme, this prototype demonstrates explainable risk scoring, consent-based verification, and secure anomaly detection workflows.

---

## 🚨 Problem Statement

With the rise of generative AI tools, synthetic media, deepfake identities, and manipulated documents are becoming increasingly common in academic environments.

Institutions face risks such as:
- Fake student identities
- Deepfake-based impersonation
- Edited academic documents
- Unauthorized digital access attempts

Most existing tools are either:
- Enterprise-heavy
- Not explainable
- Not designed for students
- Not privacy-focused

---

## 💡 Proposed Solution

AuthentiCampus AI provides:

- Consent-based verification workflow  
- AI-driven authenticity scoring  
- Synthetic anomaly detection  
- Metadata irregularity analysis  
- Explainable risk reports  
- Privacy-first design (no raw PII storage)

The system generates an **Authenticity Score** and clear **Risk Level** (Low / Moderate / High) along with explainable indicators.

---

## 🧠 How It Works

1. User provides consent.
2. File is securely uploaded.
3. AI risk engine evaluates:
   - Metadata consistency
   - Behavioral anomaly patterns
   - Synthetic signal indicators
4. Risk scoring algorithm calculates authenticity percentage.
5. Explainable report is generated.
6. Decision-ready output is displayed.
